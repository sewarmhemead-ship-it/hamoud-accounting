## 2. كشف التاجر — `فراس الشهابي.xlsx` (المرجع)

### الأعمدة الفعلية

| عمود Excel | الحقل في النظام | ملاحظة |
|------------|-----------------|--------|
| التاريخ | `entry_date` / `date` | |
| بضاعة | `goods_name` | الدفعات تُكتب هنا: «دفعة شام كاش» |
| ترسيم | `tarseem` | |
| عمال | `workers` | |
| وصل دور | `door_receipt` | |
| **اتعاب** | `clearance_fee` | ⚠️ الاسم في Excel «اتعاب» لا «تخليص» |
| سائق سوري | `syrian_driver` | |
| سائق تركي | `turkish_transport` | أحياناً مبلغ كبير (9400$) |
| مصاريف أخرى | `other_expenses` | |
| المجموع | `total_cost` | قيد-ص |
| الدفعات | `transactions.type=in` | قيد-و — **نفس الورقة** |

### المنطق

```
الرصيد = Σ(المجموع) − Σ(الدفعات)
مثال: 46,612 − 46,754 = −142$ (علينا ب142)
```

### ما ينقص في البرنامج

- [ ] **ضريبة 2%** غير ظاهرة في Excel — قد تكون مدمجة في الترسيم
- [ ] **خدمات المعبر 30$** غير عمود منفصل دائماً
- [ ] **صفوف الدفعة** في نفس الجدول (موجود في API، الواجهة تحتاج كشف موحّد)
- [ ] **صفوف مطابقة/تسوية** (200$ «مطابقة»، 15$ «شام كاش» في عمود المجموع)

## 3. المخلص — ورقتان (هامش الشركة)

### `مخلص باب الهوى.xlsx`

| | ورقة **مخلص** | ورقة **تاجر** |
|---|--------------|--------------|
| **المجموع (ص)** | 235,365 | 237,675 |
| **الدفعات (و)** | 234,680 | 230,000 |
| **الرصيد** | 685 (لنا) | 7,675 (لنا) |
| **الفرق** | — | **+2,310$ هامش** |

**الفرق = ما يأخذه التاجر − ما يُدفع للمخلص = ربح الشركة على الخدمات**

أعمدة إضافية على ورقة التاجر: `خدمات أخرى` (40$ vs 30$ على المخلص)

### ما ينقص

- [ ] **Ledger مزدوج**: `broker_cost` (تكلفة) + `trader_charge` (فاتورة)
- [ ] **ProfitService** يحسب: `trader_charge − broker_cost − driver_paid`
- [ ] عرض كشفين لنفس السيارة (زاوية مخلص / زاوية تاجر)

## 5. طازج — `وكالة عصير طازج.xlsx`

### أعمدة ورقة «تخليص» (22 عمود)

```
م | تاريخ | صنف | عدد مرسل | عدد مستلم | سائق | تاجر
| مصروف تركي 2000$ | رأس المال | قيمة طبالي
| ترسيم | عمال | تخليص | سائق | مجموع نقل
| الطرد يكلف | مجموع تكلفة | سعر بيع | مربح بيع | إجمالي مربح
```

### صيغ

```
units_received = sent − lost
cost_per_unit = (capital + transport + tarseem + workers + clearance + driver) ÷ received
profit = (sale_price − cost_per_unit) × received
```

### ورقة «اجمالي»

- مجموع عمليات + دفعات + **نقص + تلف طازج** + **باقي علينا**

### `تلف طازج.xlsx`

|------|-------|--------|------|-----|-------|
| تنك 250مل | 120 | 80 | 25+26 | 5.5 | 3,256 |

## 7. نماذج عمل إضافية

### سيارات بورتر (بيع مباشر)

```
رأس المال + جمرك + نقل + مخلص = مجموع التكلفة
→ بيع → مربح = البيع − التكلفة − أجار تراك
```

**موديل منفصل** — ليس `ShipmentService` — جدول `direct_sales` أو `porter_trucks`

### بيع طحين عزيز

```
المربح = (سعر البيع − سعر الشراء) × الوزن − أجار تراك
```

### كمية طحين (ورقة منفصلة عن الحساب)

- **ورقة كميات**: وزن + عدد + سائق + مسار (بدون مبالغ)
- **ورقة حساب**: نفس الشحنات + أرقام مالية
- = **فصل WIP الكميات عن WIP المالي** (مثل pending بدون أرقام)
